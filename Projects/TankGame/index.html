The Overview

Tank Battle 2D is a top-down shooter where players navigate a maze, dodge enemy fire, and destroy opponent tanks. The primary goal of this project was not just to make a game, but to implement scalable architecture patterns that are essential for larger projects.

Technical Deep Dive: Object Pooling

Instantiating and destroying bullets frequently causes memory fragmentation and triggers the Garbage Collector, causing frame drops. To solve this, I implemented a generic Object Pool. 

public class ObjectPool<T> where T : Component
{
    private Queue<T> pool = new Queue<T>();
    private T prefab;

    public T Get() {
        if (pool.Count > 0) {
            T instance = pool.Dequeue();
            instance.gameObject.SetActive(true);
            return instance;
        }
        return GameObject.Instantiate(prefab);
    }
}
